# LinkedIn Job Automation

A Python project that demonstrates browser automation using Selenium. The program opens LinkedIn, performs a search based on user input, extracts email addresses from visible posts using regular expressions, and sends an email with a resume attachment using Gmail SMTP.

## Features

- LinkedIn login
- Keyword based search
- Email extraction using regular expressions
- Email sending with Gmail SMTP
- Resume attachment
- Browser automation with Selenium

## Technologies

- Python
- Selenium
- WebDriver Manager
- Regular Expressions
- SMTP

## Installation

```bash
uv sync
```

or

```bash
uv add selenium webdriver-manager
```

## Run

```bash
uv run linkedin_bot.py
```
